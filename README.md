# 📱 Taze App QR Code Installer

**Scan a QR code to install Taze from:**
- 🤖 **Google Play Store** (Android)
- 🪟 **Microsoft Store** (Windows)
- 💻 **Web App** (Browser)
- 📥 **Direct Download** (APK)

## 🚀 How to Use

1. **Scan the QR code** with your phone camera
2. **Click the link** that appears
3. **Follow the instructions** to install Taze
4. **Sign in** and start managing your inventory!
